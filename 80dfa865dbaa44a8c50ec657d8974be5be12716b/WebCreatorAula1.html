>Introduction to HTML

HTML was created by Sir Tim Berners-Lee in late 1991 but was not officially released. It was published in 1995 as HTML 2.0. HTML 4.01 was published in late 1999 and was a major version of HTML.

The first web page went live on August 6, 1991. It was dedicated to information on the World Wide Web project and was made by Tim Berners-Lee.

The <!DOCTYPE> declaration represents the document type

HTML headings are defined with the <h1> to <h6> tags
<h1> defines the most important heading.
<h6> defines the least important heading

HTML paragraphs are defined with the <p> tag

:: HTML links are defined with the <a> tag

<a href="https://www.google.pt">This is a link</a>

:: HTML images are defined with the <img> tag.
The source file (src), alternative text (alt), width, and height are provided as attribute

<img src="image.jpg" alt="just a image" width="104" height="142">

HTML elements can be nested (this means that elements can contain other elements).

All HTML documents consist of nested HTML elements.

<p>This is a <br> paragraph with a <b>bold line break.</b></p>

- <br> Line break
- <ol> Defines an ordered list
- <li> Defines a list item
- <b> - Bold text
- <strong> - Important text
- <i> - Italic text
- <em> - Emphasized text
- <mark> - Marked text
- <small> - Smaller text
- <del> - Deleted text
- <ins> - Inserted text
- <sub> - Subscript text
- <sup> - Superscript text

Light Introduction to CSS { }

CSS stands for Cascading Style Sheets
CSS describes how HTML elements are to be displayed on screen, paper, or in other media
CSS saves a lot of work. It can control the layout of multiple web pages all at oncey:



:: In this example all <p> elements will be center-aligned, with a red text color:

p {
color: red;
text-align: center;
}

:: Import fonts from Google Fonts like this:

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@300&display=swap');
p{
    color: red;
    text-align: center;
    font-family: 'Roboto', sans-serif;
}
:: The background-color property specifies the background color of an element.

The background color of a page is set like this:

body {
background-color: lightblue;
}

With an external style sheet, you can change the look of an entire website by changing just one file!
Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

<!DOCTYPE html>
<html>
<head>

<link rel="stylesheet" href="mystyle.css">

</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
